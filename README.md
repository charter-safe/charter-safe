## 전세사기 예방 애플리케이션

**젊은 청춘들의 꿈을 빼앗는 전세사기를 예방하기 위해 만든 애플리케이션**

## 🔎 주요 기능

- 지도를 통해 원하는 지역의 전세 매물을 찾아보세요<br>
  ![이미지](이미지)
- 각 매물의 전세가격, 전세가율 등 여러 정보를 알 수 있어요<br>
  ![이미지](이미지)
- 그 매물의 거주자들의 평점 및 리뷰를 확인해보세요<br>
  ![이미지](이미지)
- 전세 사기 예방법을 알아보세요<br>
  ![이미지](이미지)

### 프론트엔드
| 기능 | 상세 내용 |
|---|---|
|지도 화면 및 건물 마커| • 직관적인 지도 인터페이스 제공 <br> • 특정 마커 선택하면 해당 건물의 상세 정보를 보여주는 기능 구현 |
|건물 상세 페이지| • 건물 정보를 시각적으로 보기 쉽게 구성 |
|리뷰 및 평가 기능| • 사용자 리뷰 및 별점 기능 제공 |
|디자인 및 스타일| • 녹색 계열을 사용하여 긍정적이고 안전한 이미지 강조 <br> •  |.....

### 백엔드
| 기능 | 상세 내용 |
|---|---|
|개인 데이터 보안 및 소셜 로그인| • JWT 인증을 통해 로그인용 액세스  <br> • `OAuth2.0`을 통한 Naver, Google 소셜 로그인<br> • `SecurityFilterChain`을 이용해 인증 유무에 따른 페이지 세분화|
|공공데이터 호출| • 공공데이터 API를 호출해 원하는 데이터만 추출 후 사용<br> • XML파일을 JSON 형식으로 변환 후 사용<br> • `POSTMAN`을 이용해 API 확인|
|검색 기능 제공| • `ElasticSearch`를 이용해 주소에 따른 매물 검색 가능
|AWS를 통한 CI/CD 구축| • `Github Actions`를 이용한 파이프라인 구축  <br> • `S3`에 .jar 파일을 zip 형식으로 저장<br> • `CodeDeploy`를 통해 `EC2`서버에 배포|
